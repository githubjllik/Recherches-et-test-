Quand ? - qu'est-ce qu'on a ? Qu'est-ce qui reste ? Dans cette application tout ce qui se créer obtient un QR code : produit, utilisateur À la connexion l'utilisateur peut soit taper son mot de passe ou scanner son qr code ou importer l'image de ce qr code Pour identifier un produit on peut soit chercher ou scanner son qr code Cette application peut avoir plusieurs utilisateurs Je veux que l'application propose 4 langues : - les membres peuvent être aussi promu admin si l'admin le décide, même en étant admin l'administrateur principal on ne peut pas le supprimer comme les autres, c'est lui seul qui peut décider de se retirer de ce poste On peut ajouter un nouveau produit, demandant des informations comme : -
